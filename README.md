# Project Info
Welcome to Hack4thon-Proj! This project aims to create a platform through which members of the DMV community can find cultural spaces using Google Maps data/user moderated activity.


# Tech Stack
**Frontend:**

- Framework: React
- Language: JavaScript
- Styling: CSS Modules
